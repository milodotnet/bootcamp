# Source Control

## Source control as concept and what problem it solves

- Backup
- Sync
- Revert
- Audit (who, what, when, why)
- Ability to experiment
- Parallel (branch/merge)

##  History

### 1st Gen 
Storage was expensive
No network
Lock based

SCCS (1972)
RCS (1982)

### 2nd Gen
CVS (1986), Source Safe (1995), SVN (2000), TFS(2000ands)
Centralized
Merge -> commit

### 3rd Gen
Git, Mercurial
Commit -> Merge -> Push
